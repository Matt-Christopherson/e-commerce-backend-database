# E-commerce Back End Starter Code
## Description
This project is for managers at internet retail companies that want a back end for their e-commerce website that uses the latest technologies. With this back end, any company can compete with other e-commerce companies.

## Installation
This application requires that you install Node.js, mySQL, and Insomnia.
## Usage
Open an integrated terminal and type in "mySQL -u root -p" and type in your password. Then enter "source db/schema" and then enter "quit". Now open server.js in an integrated terminal and type in "node server." Now you can navigate your database in Insomnia using links like in the demo video below.
